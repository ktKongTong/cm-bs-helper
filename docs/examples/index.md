### 谱面操作

#### 批量删除/移动谱面

开启多选模式，选择所需的谱面，进行移动或删除

![](https://lark-img.ktlab.io/yuque/0/2024/png/1419739/1705664051375-574125e1-8a61-43a1-a6b5-985963e9bc13.png#)

#### 谱面信息

![](https://lark-img.ktlab.io/yuque/0/2024/png/1419739/1705664051910-431a5ca7-6be9-4d4a-a6c2-bd0aeb6d91e1.png#)

音乐预览，会从 Beatsaver 加载歌曲预览，时长一般为十秒左右。

谱面预览，会从 Beatsaver 加载谱面，预览界面，对 pico 而言，渲染压力较大，实用性不强

删除，一键从曲包中删除谱面，**目前无任何确认操作，小心误触**

#### 曲包同步

当本地文件夹未通过本应用进行修改时，会出现数据库数据与文件数据不同的情况，此时进行同步可以修正部分错误数据。当本地数据与数据库数据差距过大时，最好的方式是重新进行扫描。

![](https://lark-img.ktlab.io/yuque/0/2024/png/1419739/1705664052322-dba07793-913f-402c-b443-73404eab9ccd.png#)

#### 谱面导入/导出

导出功能包括导出为bplist和导出为key

其中导出为 bplist可以将歌单导出为beatsaber可识别的 bplist 格式的歌单（暂不支持导入 bplist）

其中导出为key，可以将歌单导出为一串分享码，其有效期为 7 天，通过这串分享码即可进行导入操作。

#### 歌单页面

对应 BeatSaver 歌单页面，可进行基本的筛选，并进行下载。

当目标歌单为空时，进行下载会以当前管理目录自动创建对应的文件夹，并将歌单中的内容进行下载。

当目标歌单不为空时，会自动将本地歌单中不存在的谱面添加到下载任务中。

### 其他

关于部分设置内容。

#### 图片源设置

beatsaver api 国内的可访问性相对良好，谱面的下载也很流畅，但其图片加载十分缓慢，因此提供了代理图片源，其原理是采用 Cloudflare 的 worker 进行图片反代，但目前其可用性也不好，正在想办法解决中。

![](https://lark-img.ktlab.io/yuque/0/2024/png/1419739/1705664061471-232ae3c2-4a7f-46ea-8e3f-884b5f384eb3.png#)

#### 管理目录

每次曲包扫描时选中的基础目录都会被视为一个管理目录。

可以在不同的管理目录下进行切换。

#### 临时备份 & 恢复

主要用途在游戏版本降级的过程中，备份&恢复自定义谱面。

对管理目录进行临时备份之后，管理目录将不再可用，因为会从原来的文件夹移动到备份的位置。

在对游戏版本升级/降级之后，进行恢复即可。

### 下载功能

下载功能目前支持BeatSaver Playlist，单个/批量谱面下载。大部分情况下可以正常工作。

首先会将谱面 zip 包下载到当前app的私有目录下，随后解压到目标文件夹，再将 zip 包进行删除。

可以对下载任务进行暂停/恢复/删除等操作。

可能存在出错的情况，如果你愿意，可以去反馈区反馈
